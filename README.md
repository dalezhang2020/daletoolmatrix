# DaleToolMatrix\n\nShared FastAPI backend for all side-project tools.\n\n## Architecture\n\n- **1 Backend** (Railway) — this repo\n- **1 Database** (Neon PostgreSQL) — shared, each tool uses its own schema\n- **Multiple Frontends** (Vercel) — one per tool\n\n## API Structure\n\n```\n/health\n/api/imagelingo/*    ← first tool\n/api/tool2/*         ← future\n/api/tool3/*         ← future\n```\n\n## Project Structure\n\n```\nbackend/\n├── main.py              # FastAPI entry point\n├── db/connection.py     # Shared DB connection\n└── tools/\n    └── imagelingo/      # First tool\n        ├── routes/\n        ├── services/\n        └── db/\n```\n\n## Deploy\n\n```bash\nrailway up\n```\n\nSet env vars in Railway Dashboard (see .env.example).\n
